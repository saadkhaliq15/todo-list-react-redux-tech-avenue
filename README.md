# Todo List App

Simple todo list app built with React, Redux, Tailwind CSS, and Material UI.

![Todo App Preview](./todo-app-preview.png)

## Author

Saad Khaliq Khan

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
